# Golden-Moon desktop

> A desktop environment project with python!

## Overview

This project is a custom desktop environment currently under active development. It is intended for **personal use, experimentation, and learning purposes only**.

The project focuses on:

* Custom window management
* System UI components (panels, menus, settings)
* Performance and low-level control
* Clean and minimal design


## Development Status

**Work in Progress**

The desktop environment is under heavy development. Expect breaking changes, incomplete features, and experimental code.

## Supported Platforms
* Linux

## Installation
- First install the setup.sh
- run it
- wait until it completes
- clone the repo
- run desktop.py
